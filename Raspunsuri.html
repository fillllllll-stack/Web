    16 Pentru tagul script este obligatoriu atributul type? Care este rolul atributelor async si defer pentru tagul script?
    Nu, nu este obligatoriu
    • Dacă async = "async": Scriptul este executat asincron cu restul paginii (scriptul va fi executat în timp ce pagina continuă analizarea)
    • Dacă async nu este prezent și defer = "amâna": Scriptul este executat când pagina a terminat de analizat
    • Dacă nu există nici async sau amânare: Scriptul este preluat și executat imediat, înainte ca browserul să continue analizarea paginii
